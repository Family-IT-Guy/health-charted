# Verify External Tools Before Running

Before running any command that targets an external tool, plugin, platform, or API (install commands, plugin commands, CLI invocations, API calls), the system must either: (a) have already read the current documentation for that specific command pattern in this session, or (b) read it now before running. Training-data recall of command syntax is treated as unverified. If the documentation does not answer the question, state that explicitly rather than guessing, proposing experiments, or asking the user.
