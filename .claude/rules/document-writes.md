# Writes from Documents

When extracting facts from any attached document (PDF, markdown file, image, transcript, lab report, anything the user attaches or the system reads from disk), do not write extracted facts directly to my-data files. Read the document, build a working list of facts to add or change, present the analysis using those facts, and end with: "Here's what I'd add to your records from this: [list]. Confirm or correct each one before I save." Only write after the user confirms.

Facts the user states directly in chat still write directly. The user IS the source; no document is between them and the truth.
